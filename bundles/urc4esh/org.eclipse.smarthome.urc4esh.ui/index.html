Welcome to URC4ESH

URC4ESH provides you the possibility to control your smart home with user interfaces that are taylored to your personal needs and preferences. We hope you will like the improved user experience and accessibility.

Log in

About URC4ESH and OpenAPE

OpenAPEIs the Open Accessibility Personalization Extension. It gives you the possibility to get recommandations on user interfaces and user interface settings that fit best your needs and preferences. Once you have set up an Application that supports OpenAPE, these information can be used in other applications to improve the user experience and accessibility. OpenAPE is open source and the Stuttgart Media University that is hosting the project does not share any information with any third party.
